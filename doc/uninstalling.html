    • Uninstalling HPLIP


    To uninstall HPLIP you can run "make uninstall" from within the HPLIP source directory or you can run "rm -rf /usr/share/hplip" which will remove the HPLIP files.

    The HPLIP source directory is created when HPLIP is initially installed. Usually in /home/user/Desktop/hplip-version where version is the hplip version. (2.8.5 for example)
