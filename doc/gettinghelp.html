    • Getting help

    The HPLIP team provides forum-based support to users on the launchpad.net website (site run by Canonical Ltd).

    To search for answers among previously asked questions, please use this link:
    https://answers.launchpad.net/hplip/ 

    To ask a question, get help with installation or usage of HPLIP, please use this link (Launchpad.net registration is required):
    https://answers.launchpad.net/hplip/+addquestion/+login

    To report a confirmed bug (defect), please use this link (Launchpad.net registration is required):
    https://bugs.launchpad.net/hplip/+filebug/+login
    Note: Please do not report a bug (defect) if you are unsure whether or not your issue really is a bug. Please use ask a question instead.

    HPLIP also maintains a website on sourceforge.net (site run by Sourceforge, Inc.) that contains HPLIP downloads, installation help, and other resources:
    http://hplip.sourceforge.net/
